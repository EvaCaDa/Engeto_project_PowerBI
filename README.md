# Engeto projekt: PowerBI

Uživatelské jméno Discord: Eva C.D.

## Základní informace o projektu

### Dostupnost základních potravin široké veřejnosti v ČR: Průměrné mzdy, průměrné ceny potravin a jejich srovnání

Report představuje vývoj průměrných mezd v letech 2000-2021 a vývoj průměrných cen potravin v letech 2006-2018. Všechna data pocházejí z Českého statistického úřadu a byla získána prostřednictvím Národního katalogu otevřených dat. Report také obsahuje srovnání vývoje průměrných mezd a průměrných cen potravin ve srovnatelném období.
